система для управления рецептами

Я разрабатывала систему для управления рецептами. Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.

УСТАНОВКА
git clone https://github.com/aliskoz/hw2_tp
cd hw2_tp
pip install -r requirements.txt
pytest

ЗАПУСК (через среду разработки или терминал)

pyton3 hw.py
